# TEI Mapping

The TEI export script emits a compact bibliography-oriented TEI document.

| Internal field | TEI element |
| --- | --- |
| `id` | `biblStruct@xml:id` |
| `title` | `analytic/title@level="a"` |
| `creator[].name` | `analytic/author/persName` |
| `source.containerTitle` | `monogr/title@level="j"` |
| `date`, `year` | `monogr/imprint/date` |
| `identifier.doi` | `idno@type="DOI"` |
| `identifier.pmid` | `idno@type="PMID"` |

The reverse TEI importer is intentionally conservative and targets TEI generated by this project:

```bash
node scripts/convert-tei-to-internal.js exports/tei/output.xml exports/tei/imported.dataset.json
```
