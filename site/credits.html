Software

Questo sito usa il software client-side-data-explorer, creato da Lucio Pileggi (AT) SNS.IT.

The software was developed with AI-assisted coding support for implementation, refactoring, and documentation.

Data

The records displayed by this site come from the JSON dataset configured in site/data/site.config.json.

Standards

The internal model is informed by Dublin Core, TEI, DataCite, Schema.org, and CSL-JSON. Export scripts are intentionally small and dependency-free.

Runtime

The explorer runs as a static browser application with HTML, CSS, and vanilla JavaScript.
